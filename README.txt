# Stock Price Analysis & Prediction System

## Overview
This project analyzes stock price trends and predicts future prices using Python and machine learning.

## Features
- Real-time stock data using yfinance
- Moving average analysis (MA50, MA100)
- Buy/Sell signal generation
- Price prediction using Linear Regression
- Data visualization using Matplotlib

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python main.py

## Output
- Stock trend graph
- Buy/Sell signals
- Prediction results
